⚡ Быстрый старт

Как авторизоваться и войти в рабочее пространство

1

Откройте панель управления

Перейдите по ссылке вашей панели управления (Studio). Адрес выглядит как https://opera-blog-admin.sanity.studio/.

2

Войдите через Google

Нажмите кнопку входа и выберите привязанный Google-аккаунт. Ввод паролей не требуется, система авторизует вас автоматически.

3

Готово

Перед вами откроется интерфейс. Слева находится структура базы данных: Статьи, Авторы, Эпохи и другие разделы.

📝 Добавление статьи

Алгоритм публикации нового материала

1

Откройте раздел

В левом меню выберите пункт «Статьи / Оперные сюжеты».

2

Создайте документ

Нажмите на иконку карандаша или кнопку Create (Создать) в верхней части списка.

3

Заполните контент

Внесите данные в поля (заголовок, текст, обложка). Система автоматически сохраняет черновик каждую секунду.

4

Опубликуйте

Когда материал готов, нажмите зелёную кнопку Publish внизу справа. Статья появится на сайте мгновенно.

🗂️ Справочник полей

Что означает каждое поле в карточке статьи

Заголовок Обязательно
Главное название статьи. Отображается везде крупным шрифтом (например: «Кармен»).
Краткое описание Обязательно
Короткий анонс статьи для карточек на главной странице. Буквально 1-2 предложения, чтобы заинтересовать читателя.
Ссылка (URL) Обязательно
Адрес страницы. Просто нажмите кнопку Generate рядом с полем — система сама создаст ссылку из заголовка.
Главная рубрика Обязательно
Выберите из выпадающего списка основной раздел сайта для этой статьи.
Автор Опционально
Укажите создателя текста. Авторов можно предварительно завести в соответствующем разделе левого меню.
Эпоха / Страна Опционально
Дополнительные фильтры для каталогизации статьи на сайте.
Главная обложка Обязательно
Изображение для карточки и шапки статьи. Загружается с компьютера или выбирается из ранее загруженных.
Текст статьи Обязательно
Полноценный текстовый редактор. Поддерживает форматирование, списки, ссылки и вставку картинок прямо в текст.

🖼️ Работа с медиа

Загрузка обложек и изображений

📁 Загрузка файла

Нажмите Upload и выберите файл на компьютере. Либо просто перетащите картинку мышкой прямо в поле браузера.

🗄️ Библиотека

Кнопка Select открывает архив всех когда-либо загруженных картинок на проект. Очень удобно для повторного использования.

⚠️
Оптимизация: Старайтесь не загружать файлы тяжелее 3-5 МБ. Для сжатия больших фото используйте бесплатный сервис squoosh.app.

🚀 Управление сайтом

Жизненный цикл статьи

🟢 Публикация

Кнопка Publish применяет все ваши изменения и сразу отправляет их на "живой" сайт.

🟡 Редактирование

Если внести изменения в уже опубликованную статью, они не появятся на сайте, пока вы снова не нажмёте Publish.

⏸ Снятие с публикации

Нажмите стрелочку рядом с кнопкой Publish и выберите Unpublish. Статья скроется с сайта, но останется в базе как черновик.

🗑 Удаление

Через меню «Три точки» в правом верхнем углу можно нажать Delete. Удаление полностью стирает документ из базы.

💡 База знаний

Важные мелочи, которые экономят время

💡
Автосохранение: Не бойтесь случайно закрыть вкладку. Sanity сохраняет каждый введённый символ в облако. Вы ничего не потеряете, даже если выключится свет.
🔍
Поиск: В самом верху панели есть глобальный поиск. Нажмите туда и начните вводить любое слово из статьи или её заголовок — система мгновенно найдёт нужный документ.
